Androidの加速度センサーに対応した、端末をシェイクして楽しむ「おみくじアプリ」になります。

# 備考
開発中シミュレーションはエミュレータ(Nexus 5)、加速度センサー(TYPE_ACCELEROMETER)はAndroid端末実機を利用しました。
アニメーションは付属画像の回転で表現、設定にはバイブレーションon/offとアプリバージョン表示を入れています。
